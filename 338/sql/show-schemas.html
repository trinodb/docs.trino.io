10.45. SHOW SCHEMAS#

Synopsis#

SHOW SCHEMAS [ FROM catalog ] [ LIKE pattern ]

Description#

List the schemas in catalog or in the current catalog. The LIKE clause can be used to restrict the list of schema names.
